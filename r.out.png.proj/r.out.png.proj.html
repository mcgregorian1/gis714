DESCRIPTION

r.out.png.proj

NOTES

The module work only for projection available through EPSG code.

This module uses r.out.png for PNG export, r.proj for raster map reprojection and creates temporary location using g.proj. WGS84 longitude and latitude are computed from the map extent in temporary location using m.proj module.

The world file (-w flag, .wld file extension) is outputted by r.out.png while the LL WGS84 file (-l, .wgs84 file extension) which contains map extent in WGS84 longitude and latitude. Both file name extensions are added to the file name of the image.
